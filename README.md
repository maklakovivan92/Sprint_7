# Sprint 7 — API автотесты Яндекс Самокат

Учебный проект: автоматизация тестирования API сервиса **Яндекс Самокат** с использованием **pytest**, **requests**, **allure**.

## Что проверяется

### 1. Создание курьера `POST /api/v1/courier`
- курьера можно создать;
- нельзя создать двух одинаковых курьеров;
- для создания нужны обязательные поля;
- успешный запрос возвращает код `201` и тело `{"ok": true}`;
- при дубликате логина возвращается ошибка `409`;
- при отсутствии обязательного поля возвращается ошибка `400`.

### 2. Логин курьера `POST /api/v1/courier/login`
- курьер может авторизоваться;
- для авторизации нужны обязательные поля;
- при неверном логине/пароле возвращается ошибка;
- успешный запрос возвращает код `200` и тело содержит ключ `id`.

### 3. Создание заказа `POST /api/v1/orders` (параметризация)
- можно указать цвет `BLACK` или `GREY`;
- можно указать оба цвета;
- можно не указывать цвет;
- в ответе возвращается ключ `track`.

### 4. Список заказов `GET /api/v1/orders`
- в ответе возвращается список заказов.
